## Document Overview

This is Mexico's **National Cybersecurity Plan for 2025**, developed with technical support from the Inter-American Development Bank (BID). It outlines the country's cybersecurity strategy, diagnostic assessment, and roadmap for 2025-2030.

## Key Topics Covered

### Current State Assessment
- Mexico's position in regional cybersecurity context
- Cybersecurity gaps identified
- Specific threats analysis
- Cybersecurity culture assessment
- Creation of Digital Transformation and Telecommunications Agency (ATDT)

### Strategic Elements
- SWOT analysis (FODA) of cybersecurity state
- Long-term vision (2025-2030)
- Roadmap for cybersecurity projects
- Incident response capabilities (CERTs/CSIRTs)

### Metrics & Indicators
- Global Cybersecurity Index 2024 (ITU)
- Ransomware victim statistics (2019-2025)
- Incident trends 2022-2025
- Project timelines and milestones
