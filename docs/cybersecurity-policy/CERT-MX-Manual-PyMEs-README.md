# CERT-MX: Manual Básico de Ciberseguridad para MiPyMEs

**Source:** CERT-MX (Centro Nacional de Respuesta a Incidentes Cibernéticos) - Guardia Nacional de México  
**Date Added:** April 10, 2026  
**Location:** `docs/cybersecurity-policy/CERT-MX-Manual-Basico-Ciberseguridad-PyMEs.pdf`

## Overview

Official Mexican government cybersecurity manual designed for Micro, Small, and Medium Enterprises (MiPyMEs). This resource provides practical guidance for improving cyber resilience in Mexican businesses.

## Key Statistics (from the manual)

- **MiPyMEs represent 99.8%** of all companies in Mexico
- **Generate 72%** of total employment (~65 million jobs)
- **Contribute 52%** of Mexico's GDP
- **47%** of MiPyMEs are dependent on ICT/Internet for operations
- **142% increase** in cyber incidents (2015-2017), exceeding 16,900 events
- **Average loss per attack:** $50,000 USD (potentially fatal for SMEs)

## Attack Vector Breakdown

| Vector | Percentage |
|--------|------------|
| Phishing | 62% |
| Malicious Code (viruses, worms, trojans, rootkits) | 13% |
| TIC Infrastructure Vulnerabilities | 10% |
| Unauthorized Logical Access | 8% |
| DoS/DDoS | 4% |
| SPAM | 2% |
| Brute Force Attacks | 1% |
| Unauthorized Information Disclosure | 1% |

**Key Insight:** The weakest link remains **Human Resources** - phishing is the dominant attack vector.

## Manual Sections (12 Core Areas)

1. **Empleados** - Employee security awareness and training
2. **Seguridad en dispositivos móviles** - Mobile device security
3. **Seguridad en la Red** - Network security
4. **Prevención de fraude electrónico** - Electronic fraud prevention
5. **Privacidad y Seguridad de datos** - Data privacy and security
6. **Seguridad en sistemas informáticos y sitios web** - IT systems and website security
7. **Seguridad en servicios de correo electrónico** - Email security
8. **Políticas de seguridad** - Security policies
9. **Respuesta a incidentes de seguridad** - Incident response
10. **Seguridad de las operaciones** - Operational security
11. **Seguridad en mecanismos de pago electrónico** - Electronic payment security
12. **Seguridad física** - Physical security
