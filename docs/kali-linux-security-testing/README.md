## Kali Linux Tool Categories (Typical Coverage)

A Kali Linux security testing guide typically covers:

### Information Gathering
- Nmap, Recon-ng, theHarvester, Shodan
- DNS enumeration, subdomain discovery

### Vulnerability Analysis
- Nessus, OpenVAS, Nikto
- Web application scanners

### Web Application Testing
- Burp Suite, OWASP ZAP, SQLmap
- XSS, SQL injection, CSRF testing

### Password Attacks
- John the Ripper, Hashcat, Hydra
- Wordlist generation, cracking techniques

### Wireless Testing
- Aircrack-ng, Kismet, Reaver
- WPA/WPA2 auditing

### Exploitation Tools
- Metasploit Framework, ExploitDB
- Payload generation, reverse shells

### Post-Exploitation
- Privilege escalation, persistence
- Lateral movement techniques

### Forensics & Reporting
- Autopsy, Volatility
- Evidence collection, report generation
