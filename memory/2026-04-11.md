### 0b. UFW Firewall Configuration (NEW)

**Status:** Pending  
**Time Required:** 15-20 minutes  
**Priority:** MEDIUM  
**Scope:** All 3 OpenClaw nodes (111, 188, 234)

### Current Issue
- No local firewall configured on any node
- SSH exposed to entire local network (0.0.0.0:22)
- Relying only on NAT isolation for security

### Recommended Configuration
```bash
# Enable UFW with default deny
sudo ufw default deny incoming
sudo ufw default allow outgoing

# Allow SSH from trusted IPs only
sudo ufw allow from <trusted-network>/24 to any port 22 proto tcp

# Allow OpenClaw Gateway ports (localhost)
sudo ufw allow from 127.0.0.1 to any port 18789,18791,42725 proto tcp

# Allow mDNS for service discovery
sudo ufw allow 5353/udp

# Enable UFW
sudo ufw enable
sudo ufw status verbose
```
